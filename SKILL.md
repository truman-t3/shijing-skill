# 诗经创作

《诗经》是"古诗之祖":四言为主,以**赋比兴**为笔法,以**重章叠句**为节奏,以草木鸟兽起兴,朴野天然,一唱三叹。气势不在华丽堆砌,而在**形制与语感的到位**。

本技能分三层:**第一层是本文件**,从《诗经》风雅颂与名篇(关雎、蒹葭、桃夭、鹿鸣、采薇等)提炼出作诗骨架、赋比兴武器库与套语句式,足以独立写出一首合格四言诗;**第二层是 `references/` 中的四份精讲笔记**(逐篇精读、四言格律、赋比兴意象库、仿作范例),要写出媲美名篇的作品必读其中相关篇目;**第三层是 `references/originals/` 中的五篇原典**,用于沉浸语感与取法其章法。后两层缺失时照常写作,不要因缺少参考而降低标准。

## 第一步:判定类别与场合

据题材与用途,先定体式(决定语气与场景):

| 类别 | 适用题材 | 语气基调 | 常见场景 |
|------|----------|----------|----------|
| **国风** | 爱情、思念、劳作、婚嫁、离别、怨刺、乡土 | 朴素、真挚、缠绵、直抒胸臆 | 河洲、田野、桑间、道路、闺中 |
| **雅**(大/小雅) | 宴饮、赠答、颂友、感时、劝勉、家国 | 典重、雍容、恳切 | 宴席、朝堂、送行、登临 |
| **颂** | 祭祀、祝寿、颂德、庆典、追远 | 庄肃、铺张、崇隆 | 宗庙、典礼、寿筵 |

多数日常请求归**国风**;贺寿颂功归**颂**;赠友抒怀归**雅**。拿不准时问用户一句用途(自用/送人/贺礼)即可,信息足够则直接动笔。

## 第二步:按四段骨架布局

一首完整四言诗的功能骨架。短章(单章或双章)可合并,但**四项功能一个不能少**:

### 一、起兴定调(开篇不直说)
以草木鸟兽、风雨山川起笔("关关雎鸠,在河之洲""蒹葭苍苍,白露为霜"),先描他物以触景生情,再引出所咏之人/事/情。为什么:起兴是《诗经》最鲜明的呼吸——不如此便像口号。咏抽象事物(志向、节操)时,以松柏、兰、鹤等物起兴同样好用。

### 二、铺陈赋写(全诗重心,篇幅最大)
- **赋**:直叙其事其情("采采芣苢,薄言掇之"——采掇之态历历在目;"昔我往矣,杨柳依依"——行役之景直陈)。
- **比**:以彼物比此物,博喻见长("手如柔荑,肤如凝脂""如切如磋,如琢如磨")。写品格、形容、功德时尤宜。
- **细节须具体**:写到名物、情态、动作才有生命力。"思念"是空话,"青青子衿,悠悠我心""一日不见,如三秋兮"是钉子。

### 三、重章叠句(回环递进,一唱三叹)
全诗一般 **三章**,各章句式相同,**只换每句中一两个字**(多为韵脚与意象字),层层递进或反复咏叹。这是《诗经》的灵魂标志——"同构异字"而非另起炉灶。换章可换韵,韵脚以现代普通话顺口为准,不强求上古音。

### 四、收束点题(程式化收尾)
末章收束或劝勉、或祝颂、或怅望。经典收法:
- 祝颂式:"宜其室家""万寿无疆""永以为好也"。
- 怅望式:"所谓伊人,在水一方""我心伤悲,莫知我哀"。
- 反问/呼告式:"谁谓河广?一苇杭之""知我者,谓我心忧"。

## 赋比兴武器库

- **赋**(直叙):把事、情、物一行行铺开,靠动作与名物见力,忌抽象评语。
- **比**(譬喻):"如/若/犹"串起博喻;写人状物品德皆宜,以多喻叠用见工。
- **兴**(起兴):先言他物引起所咏之词;起兴之物须与正意有暗合(雎鸠和鸣→淑女君子,蒹葭霜露→伊人难即)。
- **叠字**:摹声摹状,是语感的节拍器("关关、喈喈、萋萋、依依、灼灼、悠悠、赳赳")。每章宜嵌一处。
- **双声叠韵连绵词**:"参差、窈窕、辗转、逍遥、绸缪",增其声韵之美。
- **语气助词**:句末"兮、思、止、之、矣、也、哉"生咏叹之气(尤"兮"最典型,"巧笑倩兮,美目盼兮")。
- **复沓回环**:章与章之间保留骨干句,形成回环往复。

## 套语与句式库(已内化,直接取用或化用)

以下句式采自《诗经》名篇,是四言诗的"骨血"。化用优于照抄——替换其中的名物,保留其节奏。

**起兴句式**:
- 雎鸠式:关关雎鸠,在河之洲 / 〔仿〕翩翩黄鸟,集于庭柯
- 蒹葭式:蒹葭苍苍,白露为霜 / 〔仿〕杨柳依依,雨雪其霏
- 桃夭式:桃之夭夭,灼灼其华 / 〔仿〕兰之猗猗,扬扬其香
- 物候式:春日迟迟,卉木萋萋 / 秋风肃肃,白露为霜

**赋写句式**:
- 动作复沓:采采××,薄言×之 / 〔仿〕行行重言,薄言×之
- 博喻:如×如×,如×如× / 〔仿〕如圭如璧,如松如筠
- 思念:青青××,悠悠我心 / 一日不见,如三秋兮

**重章换字句式**(同构异字):
- 所谓××,在水一方 → 二章"在水之湄" → 三章"在水之涘"
- 道阻且长 → 且跻 → 且右;白露为霜 → 未晞 → 未已
- ××之×,其叶×× → 其华×× → 其实××

**收束套语**(任选或化用):
- 祝颂:宜其室家 / 万寿无疆 / 永以为好也 / 于胥乐兮
- 怅望:我心伤悲,莫知我哀 / 瞻望弗及,泣涕如雨
- 呼告反问:谁谓河广?一苇杭之 / 知我者,谓我心忧

**四言节奏模板**:二二拍("关关/雎鸠,在河/之洲")为主;可间以带"兮"的杂言("巧笑/倩兮,美目/盼兮")调节奏。自造四言先定二二拍,再于关键处嵌叠字与语助,声韵自生。

## 输出格式

- 标题:《××》(风/雅/颂 · 主题),或《拟××》《××歌》。
- 篇幅:默认三章、每章四至八句(短章双章亦可);用户要求铺陈时可加章至五章以上。
- 正文以空行分章,不加小标题(诗是一气呵成,不是报告)。
- 交稿后附 **【简注】**(难字词 3–6 条)与 **【译文】**(逐章白话意译),可附一句话说明取法哪篇路数;不逐句注释。
