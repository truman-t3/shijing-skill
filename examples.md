# 示例输出（描述示例）

本文件展示《诗经创作》skill 在不同体式与题材下的输出，供使用时参照风格与格式。
每首均含：标题（体式·主题）、正文（三章重章叠句，隔句押韵）、【简注】、【译文】、【取法】。

> 原则：**只化用手法与意象，不照抄《诗经》原句**；现代题材用文言句法包裹，不破四言古雅。

---

## 示例一 ·《夜氓》（国风·戏仿 · 熬夜写代码）

> 形式越庄重、对象越琐碎，反差越好笑——验证 skill「古雅包装现代对象」能力。

**第一章**
灯荧荧兮，夜将半矣。
案牍如山，独坐目眴。
Code 盈尺，键声漫漫。
寤寐营营，其劳何叹。

**第二章**
窗冽冽兮，月落峦矣。
理路转寒，布格难刊。
穷搜行间，如篦衣蟏。
Bug 潜潜，其劳何酸。

**第三章**
星沉沉兮，钟鼓成矣。
灵台既陈，绿章既明。
偃卧少安，劳人其宁。
勖哉诸君，免乎夜氓。

**【简注】**
- 荧荧：灯火微明貌。眴（xuàn）：目摇，谓久视而眩。
- 案牍：此指需求文档/工单。布格：bug 之音译。刊：削改、除错。
- 篦（bì）：密梳。衣蟏（xiāo）：衣缝。喻于代码中逐行搜错，如篦衣捉虱。
- 灵台：心/机枢，此指 IDE 与云端构建。绿章：测试全过之报（绿）。
- 勖（xù）哉：勉之。夜氓：夜中为人役使之码农（氓，民也）。

**【译文】**
灯火微明啊，夜已过半。案上文书堆如山，独坐眼花。代码盈尺，键盘声连绵。醒睡都忙活，这辛劳叹个啥。
寒窗凛冽啊，月落山脊。逻辑转冷，bug 难除。遍搜代码行间，如篦衣缝捉虱。bug 暗潜，这辛劳何其酸。
星沉天将晓啊，构建已成。代码既付灵台，绿报既明。躺下稍安，劳人得宁。勉诸君，莫落得夜氓这般下场。

**【取法】** 兴比合一于「灯荧荧/窗冽冽/星沉沉」三章换景递进；二章以「篦衣捉虱」为**比**，讥搜 bug 之琐；三章收以「勖哉」呼告——得国风戏仿「大词小用」之趣。

## 示例二 ·《松柏》（雅·咏物明志）

> 庄重对象配庄重形式，验证 skill「严肃仿古」正格。

高山苍苍，松柏其芳。
寒风烈烈，不改其刚。
冰霰皑皑，凝姿愈庄。
君子如斯，贞心允臧。

崇冈岩岩，松柏其安。
飙风发发，不改其端。
雷殷殷兮，柯叶弥繁。
岁寒知后，乃见贞完。

峻极茫茫，松柏其苍。
千寻挺秀，独傲雪霜。
贞心似石，历久弥昌。
勉尔君子，永矢弗忘。

**【简注】**
- 苍苍：深青色，状山色幽远；芳：松柏清芬，喻德馨。
- 皑皑：雪白貌；凝姿：凝固不动之姿，言不为寒所动。
- 岩岩：山高峻貌；飙风：暴风；发发（bō）：风疾之声。
- 殷殷：雷声；贞完：坚贞完固。
- 峻极：山极高；千寻：极言其高（寻，八尺）；弥昌：愈盛。
- 永矢弗忘：永誓不忘（矢，誓）。

**【译文】**
高山幽远啊，松柏自有清芳。寒风凛冽，它不改刚直。冰雪皑白，姿容更显庄重。君子像它，贞正之心信然可嘉。
高冈峻拔啊，松柏安然自立。暴风疾疾，它不改端直。雷声殷殷，枝叶更繁。到岁寒之后，才见它坚贞完固。
极顶苍茫啊，松柏苍翠。高达千寻，独傲霜雪。贞心如石，日久愈盛。勉勖君子，永誓不忘此志。

**【取法】** 高山／崇冈／峻极三章换景递进起兴；以「不改其刚／端」「乃见贞完」赋写坚贞；末章「贞心似石」为**比**，收以「勉尔君子，永矢弗忘」**呼告**——得小雅「雍容恳切、以物明志」之体。

## 示例三 ·《夜人》（国风·戏仿 · 熬夜做设计）

> 同戏仿路线，对象是设计师改稿——甲方改稿梗，反差生笑。

**第一章**
灯荧荧兮，夜将分矣。
素幅盈案，独运匠心。
设色漫漫，点染纷纷。
寤寐营营，其劳何殷。

**第二章**
窗冽冽兮，月落津矣。
主顾数更，三易其文。
毫厘拣尽，如篦衣纹。
稿积壅壅，其劳何醺。

**第三章**
星沉沉兮，钟漏尽矣。
灵台既呈，彩章乃新。
偃卧少安，劳人其臻。
勖哉同辈，免作夜人。

**【简注】**
- 荧荧：灯火微明貌。分：半（夜将分，夜过半）。
- 素幅：白绢，此指画布/设计稿。设色：敷彩，此指配色。
- 冽冽：寒也。津：津渡，言月落水滨、夜已深。主顾：甲方。数更：屡次更改。
- 毫厘：极细微处，此指像素/间距。篦（bì）：密梳。衣纹：衣缝。喻于稿中逐处搜检，如篦衣捉缕。
- 壅壅：堆积貌，言改稿层累。
- 灵台：心/机枢，此指屏幕与渲染输出。彩章：彩色定稿。
- 勖（xù）哉：勉之。夜人：夜中为人役使之设计师（仿《夜氓》「夜氓」）。

**【译文】**
灯火微明啊，夜已过半。画布堆满案头，独自运使匠心。配色连绵，点染纷纷。醒睡都忙活，这辛劳何其殷勤。
寒窗凛冽啊，月落津渡。甲方屡次改稿，三易其文。毫厘之处拣遍，如篦衣缝寻缕。改稿层累堆积，这辛劳何其疲极。
星沉天将晓啊，钟漏将尽。屏幕既呈，彩稿乃新。躺下稍安，劳人得其所。勉诸同辈，莫作了这夜中人。

**【取法】** 兴比合一于「灯荧荧／窗冽冽／星沉沉」三章换景递进；二章以「篦衣寻缕」为**比**，讥搜检细节之琐；三章收以「勖哉同辈，免作夜人」**呼告**——得国风戏仿「大词小用」之趣。
