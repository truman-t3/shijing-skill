# 四言格律 —— 句式、押韵、叠字与节奏

《诗经》非格律诗(无平仄拘束),但有稳定的声韵习惯。照此写,自然古雅。

## 一、句式:四言二二拍

四言诗以**二字一顿**为主,读来两拍:"关关/雎鸠,在河/之洲""蒹葭/苍苍,白露/为霜"。

- 基准句:四字。每章四至八句最常见。
- 调节奏:可间以二言("于嗟!/麟兮!")、五言,或带"兮"的杂言("巧笑/倩兮,美目/盼兮")——但整体四言骨架不可散。
- 避忌:勿让七言律句混入("平平仄仄平平仄"那种),那是唐人格律,非诗经体。

## 二、押韵:隔句押、换章可换韵

- 主要押法:**偶句押韵**(第 2、4 句),首句可押可不押。
- 换章换韵:三章可三换韵(如蒹葭:霜/方/长 → 晞/湄/跻 → 已/涘/沚),每章内部自押即可。
- 韵部:以**现代普通话**读来顺口为准,同章内韵母相近即可;**不必考订上古音部**(顾炎武、王力之学中古音非必需)。
- 通押放宽:支思、鱼虞、真文等邻近韵可适当通押,以自然为度。

## 三、叠字:声韵的节拍器

叠字(重言)是诗经体最显眼的声韵标记,摹声摹状,每章宜嵌一处。

- 摹声:关关、喈喈、呦呦、雝雝、坎坎、丁丁、萧萧。
- 摹草木:灼灼、萋萋、苍苍、依依、蓁蓁、离离、菁菁。
- 摹情态:悠悠、忉忉、悄悄、赳赳、穆穆、翙翙。
- 摹天时:霏霏、迟迟、凄凄、湑湑、湛湛。
- 摹众盛:济济、洋洋、汤汤、赫赫、皇皇、锵锵。

用法:同一意象跨章可用不同叠字递进(苍苍→萋萋→采采),显时间流转。

## 四、双声叠韵与连绵词

- 双声(声母同):参差、辗转、踌躇、须臾。
- 叠韵(韵母同):窈窕、逍遥、绸缪、须臾、蟋蟀。
- 用法:点缀于句中增声韵之美,不宜堆砌;一首一处分寸最佳。

## 五、语气助词:咏叹之气

句末或句中虚字,生一唱三叹之韵:

- 兮:最典型,"巧笑倩兮,美目盼兮""坎坎伐檀兮,置之河之干兮"。
- 思:语助足句,"南有乔木,不可休思""今我来思,雨雪霏霏"。
- 止:足句兼略表时,"亦既见止,亦既觏止""岁亦莫止"。
- 之 / 矣 / 也 / 哉 / 只 / 且:调节语气与节奏。

用法:国风多用"兮",雅颂多用"思/止",可据类别择用;但全篇宜统一一种主调,勿杂乱。

## 六、节奏自查(写完逐条过)

1. 是否四言二二拍为主?有无七言律句混入?(有则改)
2. 偶句是否押韵?换章是否换韵自然?(不押则顺口读一遍改韵脚)
3. 每章是否至少一处叠字?(无则补一处)
4. 语气助词是否统一调性?(杂则择一)
5. 三章是否"同构异字"、只换 4–6 字?(另起炉灶则改回平行结构)
6. 有无现代词、网络语、白话直露?(有则换古雅名物)
